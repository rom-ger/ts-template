1. Создать папку src/actions, в ней создать файл jsonplaceholderActions.ts, в этом файле написать класс со статическими методами с обращением к апи
апи смотреть здесь https://jsonplaceholder.typicode.com/ в разделе Resources, шесть апи = шесть статических методов
2. Для каждого метода описать интерфейсы возвращаемых данных
3. в файле src/class/myFirstClass.ts добавить метод getApi, который должен вызвать все методы из jsonplaceholderActions.ts и получив все данные из них, объединить их в один результирующий объект вида
{
    posts: ...
    comments: ...
    albums: ...
    ...
}
получившийся объект вернуть, как результат выполнения функции
4. Для этого создать ветку в гите, всё в ней реализовать и сделать пул реквест
